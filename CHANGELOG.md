# Changelog

## [0.1.0] - 2025-05-08

### ✨ Primera versión
- Widget `MiPhoneInput` con selector de país integrado
- Modelo `PhoneNumber` con número completo y validación básica
- Modelo `Country` con 70+ países, banderas emoji y códigos de marcado
- `CountryPickerDialog` con búsqueda en tiempo real
- Selector personalizable mediante `selectorBuilder`
- Soporte para lista de países restringida
- Callbacks `onInputChanged` y `onInputValidated`
- Tests unitarios para modelos
